Assignment Title: Earthquake Map

What class was this project for? App Academy

The Earthquake map uses United States Geological Survey API and Google Maps. The map will show any seismic activity that happens throughout the day. The earthquake map is an interesting website for people to see the real-time seismic activity thats a magitude 2.5 and above from the past few days. The surprising part of the project is how much activities occur daily throughout the world.
